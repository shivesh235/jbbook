Pandas#

Pandas is a powerful and widely used open-source Python library for data manipulation and analysis. It provides flexible and efficient data structures, primarily DataFrames and Series, to handle structured data such as tabular data (similar to spreadsheets or SQL tables) and time series data.

Key features of Pandas include:

  • Data Wrangling: Tools for cleaning, transforming, and reshaping data.

  • Indexing and Slicing: Easy access to rows and columns using labels or positions.

  • File I/O: Importing and exporting data in formats like CSV, Excel, SQL, JSON, and more.

  • Statistical Analysis: Descriptive statistics, aggregation, and group operations.

  • Handling Missing Data: Identifying, imputing, or dropping null values.

Pandas is built on NumPy, which provides the foundational structures and performance for numerical operations. It’s an essential tool for data scientists and analysts, making complex data manipulation tasks much simpler and faster.

import pandas as pd

2 main datatypes#

Pandas provides two core data structures:

  1. Series:

    • A one-dimensional labeled array capable of holding any data type (integers, floats, strings, etc.).

    • It is similar to a Python list or a NumPy array but includes indexing, allowing for easy data retrieval and alignment.

  2. DataFrame:

    • A two-dimensional labeled tabular structure with rows and columns, where each column can hold different data types (like a spreadsheet or SQL table).

    • It is made up of multiple Series objects.

Both structures are versatile and efficient for handling real-world data operations like filtering, aggregation, and transformations.

Import files#

Pandas provides powerful tools to import and work with various file formats. The most commonly used functions include:

  1. CSV Files:

    • pd.read_csv("file.csv"): Reads a CSV file into a DataFrame.

    • Supports options like sep, header, index_col, usecols, etc.
      Example:

    df = pd.read_csv("data.csv")

  2. Excel Files:

    • pd.read_excel("file.xlsx", sheet_name="Sheet1"): Reads Excel files.
      Example:

    df = pd.read_excel("data.xlsx")

  3. JSON Files:

    • pd.read_json("file.json"): Reads JSON data into a DataFrame.
      Example:

    df = pd.read_json("data.json")

  4. SQL Databases:

    • pd.read_sql(query, connection): Reads data from SQL databases.
      Example:

    import sqlite3
conn = sqlite3.connect("database.db")
df = pd.read_sql("SELECT * FROM table", conn)

  5. Other Formats:

    • Pandas supports additional formats like Parquet (pd.read_parquet), HDF5 (pd.read_hdf), and more.

Each function comes with flexible parameters to handle large datasets, custom delimiters, missing values, and encoding issues.

Describing Data#

Pandas DataFrames offer several attributes and methods to describe and understand the dataset:

Key Attributes:

  • .shape: Provides the number of rows and columns.

  • .columns: Returns the column names.

  • .index: Returns the row indices.

  • .dtypes: Shows the data type of each column.

Descriptive Methods:

  • .head(n): Displays the first n rows (default is 5).

  • .tail(n): Displays the last n rows (default is 5).

  • .info(): Summarizes the DataFrame, showing column types, non-null counts, and memory usage.

  • .describe(): Provides summary statistics for numeric columns (e.g., mean, median, min, max).

  • .value_counts(): Counts unique values in a Series.

  • .isna() / .isnull(): Identifies missing values.

  • .corr(): Computes pairwise correlation of numeric columns.

These tools help quickly explore and understand the structure, content, and quality of your dataset.

Viewing and Selecting data#

Some common methods for viewing and selecting data in a pandas DataFrame include:

  • DataFrame.head(n=5) - Displays the first n rows of a DataFrame (e.g. car_sales.head() will show the first 5 rows of the car_sales DataFrame).

  • DataFrame.tail(n=5) - Displays the last n rows of a DataFrame.

  • DataFrame.loc[] - Accesses a group of rows and columns by labels or a boolean array.

  • DataFrame.iloc[] - Accesses a group of rows and columns by integer indices (e.g. car_sales.iloc[0] shows all the columns from index 0.

  • DataFrame.columns - Lists the column labels of the DataFrame.

  • DataFrame['A'] - Selects the column named ‘A’ from the DataFrame.

  • DataFrame[DataFrame['A'] > 5] - Boolean indexing filters rows based on column values meeting a condition (e.g. all rows from column ‘A’ greater than 5.

  • DataFrame.plot() - Creates a line plot of a DataFrame’s columns (e.g. plot Make vs. Odometer (KM) columns with car_sales[[“Make”, “Odometer (KM)”]].plot();).

  • DataFrame.hist() - Generates histograms for columns in a DataFrame.

  • pandas.crosstab() - Computes a cross-tabulation of two or more factors.
